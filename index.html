📖 使用说明

📝 注册语音

1. 上传音频文件（支持 MP3、WAV、FLAC 等格式）

2. 输入语音 ID（用于标识该语音）

3. 输入该音频对应的文本内容

4. 点击"注册音频元数据"按钮

🎭 克隆语音

1. 选择已注册的语音 ID

2. 输入要转换的文字内容

3. 调整语速和输出格式（可选，默认 MP3）

4. 点击"生成语音"按钮

🔧 API 端点

语音合成：POST /v1/audio/speech

注册语音：POST /v1/audio/voices

获取语音列表：GET /v1/audio/voices

传统生成端点：POST /generate

📋 示例请求

# 获取所有已注册语音列表 curl http://localhost:8080/v1/audio/voices # 注册语音 # 注意：s2.cpp 需要 file (音频)、reference_text (对应文本)、voice_id curl -X POST http://localhost:8080/v1/audio/voices \ -F "voice_id=my_voice" \ -F "reference_text=这段音频对应的文本" \ -F "file=@prompt.wav" # 生成语音 (OpenAI 兼容格式，支持 mp3/wav/pcm) # 默认格式为 mp3 curl -X POST http://localhost:8080/v1/audio/speech \ -H "Content-Type: application/json" \ -d '{ "input": "你好，这是测试", "voice": "my_voice", "response_format": "mp3" }' \ --output output.mp3 # 传统生成端点 (支持更多参数) curl -X POST http://localhost:8080/generate \ -F "text=要合成的文字" \ -F "voice=my_voice" \ -F "reference_text=参考音频文本" \ -F "reference_audio=@ref.wav" \ -F 'params={"stream": true, "low_latency": true}' \ --output output.wav
